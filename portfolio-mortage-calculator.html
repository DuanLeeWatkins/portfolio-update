Mortgage Calculator App

Beatme2 is a beat/instrumental audio file management application. Users can upload beats/instrumentals to store online and have access to them anywhere and anytime.

Tech used: JavaScript, React.js, Supabase, and MUI. I used React to build the user interface. Supabase will handle a lot of the data management and other backend work. I used MUI for the styling. I used git and github for version control and the app will be hosted on Railway.

I definitely enjoyed working on this app because it is something that the music producer community could use. I know because I am apart of that community. Having a cloud based file management app that is tailored for music creatives would provide a great alternative to other file management apps like Dropbox or Google drive. Creating this app has helped me understand that it takes a lot to build an app. If I had to restart building this app I would plan the development of the app better before starting on the code.
